Sobre o Projeto

Proposta inicial

Esse projeto é um trabalho academico, da materia de Web Avançado, com o professor Wanderson Muniz de Santana. Onde os alunos foram livres para criar do zero um projeto fullstack, utilizando algumas tecnologias já pré definidas.

Stack

O projeto atualmente possui poucas libs, mostrando que com pouco ferramental ainda dá pra fazer um bom trabalho. As libs utilizadas são:

  • Flask
  • SQLAlchemy
  • Playwright
  • Beautiful Soup 4
  • Flask-WTF

Criadores

O projeto foi criado por apenas um integrante, Rafael Venturini Dipalma

Sobre o funcionamento geral do Projeto

O Projeto atualmente se baseia em um webscraper de dados sobre o Porto Central, usando ferramentas como Playwright, Beautiful Soup 4 e requests diretos à sites que dizem sobre o assunto.

O Scraper

O Projeto atualmente coleta dados dos seguintes sites:

  • Porto Central: Coleta direta pela api do Word Press
